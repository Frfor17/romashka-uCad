привет, это дока о том, как работать с серваком

Сначала надо зайти в нужную папку на сервере, то есть ну, в папку репозитория на сервак, затем надо вкинуть команду
команду для активации виртуального окружения, чтобы все модули и библиотеки работали


source env/bin/activate

потом надо вкинуть штуку, чтобы подтянулись последние комминты и обновения с ветки мейн из репозиторий

git pull origin main 

затем, мы можем запустить бота одним из нескольких способов. Первый чисто для теста, второй это чтобы он всегда работал

Первый способ(python3 потому что на сервере убунта и на всех серверах вроде бы чисто python3 пашет тока):

python3 tg_bot.py

ВТОРОЙ СПОСОБ. Он врубает фоновый процесс и в целом логи отправляются в специальный фаайл, ну короче способ
это способ чтобы врубить бота и им пользовалисьы

nohup python3 tg_bot.py &

чтобы выключить бота(это не точно что так работает), надо найти  и убить процесс этот фоновый

ps aux | grep tg_bot.py  # Найти PID
kill <PID>               # Заменить <PID> на найденный номер
